Botones

Los botones pueden ser creados con tres tipos de etiquetas las cuales son los hipervinculos 'a', 'button', e 'input';
Primero los botones con hipervinculos. (< a href="#"> valor < a> lleva de cierre ).

Boton

Boton con la etiqueta button. (< button >valor< button>)

Boton con la etiqueta input. (< button >valor< button>). Puede tomar varios valores como reset, submit, text, etc..

Tambien se pueden crear grupo de botones de manera horizontal o vertical, primero los horizontales esto se hace en una etiqueta div con un attr llamado data-controlgroup(para el grupo) y data-type(para la posición).
